W³ - Weekly War Room & Writing
个人知识管理系统 - 学习战报 + 日记博客 + 数据导出

🌟 项目简介
W³ 是一个完全本地化的个人知识管理工具，支持：

📅 周战室：设定目标、管理任务、跟踪进度
📓 日记博客：写作、编辑、分类管理
📤 导出中心：JSON/Markdown 导出与备份
💾 本地存储：数据完全保存在浏览器，无需服务器
✨ 核心功能
🎯 周战室（War Room）
自定义本周标题
添加/删除目标
任务清单管理
专注度评分（1-10）
自动计算完成率
📝 日记博客（Journal）
Markdown 格式支持
文章分类（学习/生活/随笔/技术/思考）
搜索功能
实时预览
编辑/删除管理
📊 导出中心（Export）
导出 JSON（完整备份）
导出 Markdown（易读格式）
导入 JSON（数据恢复）
清空数据（危险操作）
🚀 快速开始
安装依赖
bash
npm install
运行开发服务器
bash
npm run dev
构建生产版本
bash
npm run build
📦 技术栈
框架：React 18
构建工具：Vite
样式：Tailwind CSS
图标：Lucide React
存储：localStorage

💡 使用说明
首页 Dashboard：查看数据概览，快速跳转功能
周战室 War Room：设定本周目标，管理任务清单
日记博客 Journal：写作、编辑文章
导出中心 Export：备份数据，导入/导出
